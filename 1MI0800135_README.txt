Съдържание на архива:
1. Components - преизползваеми графични елементи
2. Controllers - класове, играещи ролята на контролери от MVC архитектурата
3. Database - скриптови файлове за миграциите на базата данни
4. Examples - примерна таблица със студенти за експорт
5. Middleware - допълнителна междинна бизнес логика, действаща като посредник между маршрутизацията и контролерите.
6. Models - класове, моделиращи данните от базата 
7. Pages - страниците в приложението
8. Scripts - код с полезни скриптове, имащи разнообразни цели
9. Services -  класове с бизнес логика 
10. Static - статични файлове
11. Utils - полезни функции
12. .env.example - примерни стойности за необходимите променливи на средата
13. .htaccess - пренасочване на заявките към единствената входна точка на приложението
14. config.php - зарежда конфигурационния файл
15. index.php - регистриране на пътища
16. router.php - преизползваема логика за регистриране на пътища
17. styles.css - общи стилове за целия проект


Инструкции за подкарване на проекта:
1. Инсталиране на  средата XAMPP, версия 8.12.2-0 или по-висока
2. Разархивиране на кода в директорията htdocs на XAMPP (или в някоя нейна поддиректория)
3. Създаване на .env файл като се следва образеца от .env.example
4. Стартиране на Apache Web Server и MySQL Database.
5. Пускане на миграциите на базата с php ./scripts/run_migrations.php -m -a
6. Пускане на сийд за базата с php scripts/seed_database.php seed
